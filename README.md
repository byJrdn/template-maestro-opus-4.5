**Project Overview**

Template Maestro (Opus 4.5) — a browser-based template management and autofix utility used to map, validate, and export template data.

**Quick Start**

- Open `index.html` or `admin.html` in a browser for the UI.
- Serve the project locally (recommended) with Python:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

**Files of Note**

- `index.html`, `admin.html` — entry pages.
- `js/` — application scripts (e.g., `template-manager.js`, `validation-engine.js`).
- `css/` — stylesheets.
- `_` — SQL and docs.
